### Week 1 Results Achieved:

- Understand basic AWS services and service groups:
  <br> + Compute (EC2)
  <br> + Networking (VPC, Route 53)
  <br> + Cost Management (AWS Budgets, Cost Explorer)
  <br> + Security (IAM, Security Groups)

- Get familiar with AWS Management Console and know how to find, access, and use services from the web interface

- Learn basics of drawing AWS architecture on draw.io for system design

- Install and configure AWS CLI on computer including:
  <br> + Access Key & Secret Key
  <br> + Default Region
  <br> + Profile configuration

- Use AWS CLI to perform basic operations such as:
  <br> + Check account information & configuration
  <br> + Get list of regions
  <br> + View EC2 services
  <br> + Create and manage key pairs
  <br> + Check running service information

- Master IAM concepts and practice:
  <br> + Create IAM users
  <br> + Create IAM groups
  <br> + Create IAM roles
  <br> + Assign permissions
  <br> + Switch IAM Roles

- Practice managing AWS costs with AWS Budgets:
  <br> + Create Cost Budget
  <br> + Create Usage Budget
  <br> + Create Savings Plans Budget

- Understand VPC and its components:
  <br> + Subnets (Public & Private)
  <br> + Route Tables
  <br> + Internet Gateway
  <br> + NAT Gateway
  <br> + Security Groups

- Deploy EC2 instance with VPC configuration and SSH connection

- Learn VPN configuration:
  <br> + Site-to-Site VPN
  <br> + Configure VPN using strongSwan
  <br> + AWS Transit Gateway
- Understand Infrastructure as Code and automate VPC deployment
