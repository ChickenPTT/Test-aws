# "AWS Community Day Workshop" Learning Summary

- Share best practices in modern application design and effective learning methods.

- Introduce advanced AI tools and models to help optimize the entire development lifecycle.

### Highlights

#### Real-World Project from the 36H Code Competition

- Shared the journey of overcoming time pressure from the GenAI Engineers team (**Thao Nguyen, Mai Nguyen, Uyen Le**).
- How to break down a large problem into smaller tasks and apply Generative AI to speed up coding and quickly complete a working prototype.

#### AI Trends and Mindset

- **AI-first Mindset**: Reshaping how problems are solved by placing AI at the center of operational processes and system optimization to achieve the highest performance.
- **Second AI Brain**: The concept of building an AI-powered "auxiliary brain" to help people store knowledge, perform in-depth data analysis, and make more accurate decisions.

#### The "Track Line Content Mean" Thinking Framework

Modeling a method for delivering coherent and persuasive content through a 4-step core structure:

- **Goal**: Clearly define the final objective to be achieved.
- **Situation**: Objectively assess the current context and circumstances.
- **Constraints**: List limitations, barriers, and technical/resource constraints.
- **Evidence**: Provide data, figures, and real evidence to support the argument.

#### Configuring and Optimizing Large Language Models (LLM Settings)

Understanding the deeper meaning of the settings used to strictly control how an LLM operates, its creativity, and its safety:

- _Model selection_: Criteria for choosing a model suited to both the economic and technical requirements of the problem.
- _Temperature_: The mechanism for adjusting the creativity and randomness of the response.
- _Max tokens_: Controlling and limiting output length to optimize cost.
- _Role/system prompts_: Techniques for defining the AI's role, context, and response style.
- _Fine-tuning/adapters_: Methods for customizing, or "fine-tuning," a model based on an organization's specific data.
- _Safety & moderation_: Filters set up to remove sensitive or harmful content.

#### Performance Optimization with Amazon CloudFront

- A deep dive into the core concepts and content delivery network (CDN) architecture of AWS CloudFront settings.
- Implementing **best practices** such as caching strategies and edge security to reduce latency and save bandwidth for the system.

### Key Takeaways

#### Teamwork and Hands-On Skills

- **Optimal role division**: Understood the importance of recognizing each team member's strengths in order to distribute work effectively on projects with strict time limits (such as a 36-hour hackathon).
- **Leveraging the power of AI**: Learned how to integrate GenAI directly into the workflow to handle repetitive tasks, freeing up resources for logical thinking.

#### Architecture and Systems Design Thinking

- Applied the **Goal → Situation → Constraints → Evidence** framework proficiently when presenting technical solutions, making it easier to persuade stakeholders.
- Gained a solid grasp of LLM configuration parameters to fine-tune AI applications for higher accuracy and reduce hallucination.
- Improved infrastructure optimization thinking by correctly configuring Amazon CloudFront and understanding how a global content delivery network operates.

---

### Applications to Work and Study

- **Building a personal Second Brain**: Deploying AI tools to help with note-taking, summarizing study materials, and managing ongoing tech projects.
- **Prompt optimization**: Applying _System Prompts_ configuration techniques and adjusting _Temperature_ appropriately for specific coding tasks (e.g., lowering temperature for accurate code, raising it for generating test-case ideas).
- **Practicing Cloud Architecture**: Experimenting with Amazon CloudFront configurations for school assignments or personal projects to speed up page loading and static asset delivery.
- **Improving presentation skills**: Using the _Track Line Content Mean_ structure for project progress reports or presentations to a school/company board.
